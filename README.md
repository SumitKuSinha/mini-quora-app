# Mini Quora App

A simple CRUD-based web application inspired by Quora.

## Features
- Create posts
- Edit posts
- Delete posts

## Tech Stack
- Node.js
- Express.js
- EJS

## Purpose
Built to practice backend development, routing, and CRUD operations.

This project was built as a learning exercise.
